# Proof: identity-based baseline exclusion (safety.viz#91)

The peak computation excludes the resolved baseline record **by identity**, not by
study day `> 0`. A participant with no day-0 record falls back to an unscheduled
baseline visit; under the old `day > 0` rule that record counted as its own
on-treatment value, so the peak could never fall below baseline — erasing the exact
"bar drops below baseline" signal the ALT waterfall (Amirzadegan 2025 Fig 5) is built
on. This script reduces `site/data/adbds.csv` under both rules and enumerates the
rule-target population — every participant with no day-0 record — marking which of
them see their composite values move.

## Summary

| Metric                                                         | Value                      |
| -------------------------------------------------------------- | -------------------------- |
| Distinct demo participants                                     | 318                        |
| Participants with no day-0 record — the rule-target population | **24**                     |
| ...of which `CLD-` participants                                | **0**                      |
| Composite population (old `day > 0`)                           | 295 included / 23 excluded |
| Composite population (new identity)                            | 293 included / 25 excluded |
| ...of the 24 affected, those whose composite values move       | **11**                     |

Only the 24 participants with no day-0 record can differ between the two rules: a
participant WITH a day-0 record resolves its baseline to day 0, and `day > 0` is then
identical to "exclude the baseline by identity, floor at day 0". Of the affected
24, the baseline was already dominated by a later on-treatment value for
13, so their composite values do not move; the remaining 11 do
(9 value changes, 2 newly excluded).

This matches issue #91's "24 of 318 demo participants who have no day-0 record": the
rule targets all 24; the composite/waterfall signal moves for the subset whose own
baseline was their peak.
